

# 🤖 TyumenChatBot — анонимный Telegram-бот для общения

**TyumenChatBot** — это Telegram-бот, который позволяет людям общаться анонимно, искать собеседников по интересам и обсуждать городские события.  
Работает с реальными пользователями.

---

## 🚀 Возможности

- ✅ Анонимный чат «1 на 1» (рандомный поиск собеседника)
- ✅ Команды: `/start`, `/search`, `/stop`, `/next`
- ✅ Хранение состояний и диалогов в MySQL
- ✅ Чёрный список (пожаловаться / заблокировать)
- ✅ Статистика использования (количество чатов, активность)
- ✅ Inline-клавиатуры и callback-запросы
